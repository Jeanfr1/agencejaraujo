## 🚀 Overview

Agence Jaraujo is a cutting-edge digital innovation agency that combines creative design with technical excellence. Our platform showcases our services, projects, and commitment to delivering exceptional digital experiences.

## ✨ Features

- 🎨 Modern, animated UI with Framer Motion
- 🌐 Multilingual support (English, French, Portuguese)
- 📱 Fully responsive design
- 🎥 Dynamic video backgrounds
- 📬 Interactive contact form
- ⚡ Built with Next.js 13 and TypeScript
- 🎯 SEO optimized
- 🔥 Stunning animations and transitions

## 🛠️ Tech Stack

- **Framework:** Next.js 13
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Language:** TypeScript
- **Internationalization:** next-international
- **Form Handling:** React Hook Form
- **Email Service:** EmailJS

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/agence-jaraujo.git
   ```

2. Install dependencies:
   ```bash
   cd agence-jaraujo
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🌍 Internationalization

The website supports multiple languages:
- 🇺🇸 English (default)
- 🇫🇷 French
- 🇧🇷 Portuguese

Language switching is handled automatically based on the URL prefix:
- English: `/en/*`
- French: `/fr/*`
- Portuguese: `/pt/*`
