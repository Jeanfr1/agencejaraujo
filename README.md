## üöÄ Getting Started

```bash
# Clone the repository
git clone https://github.com/Jeanfr1/agencejaraujo.git

# Navigate to the project directory
cd agencejaraujo

# Install dependencies
npm install

# Run the development server
npm run dev

# Open http://localhost:3000 in your browser
```
